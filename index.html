Вступление

Здравствуй, дорогой читатель. Если ты читаешь это, то значит тебе интересно современное программирование. Здесь ты узнаешь основы JavaScript.

JavaScript - это ...

... один из самых популярных языков программирования, который заставляет выглядеть веб-страницы живыми. Раньше в давние времена эти страницы были статичными, плоскими, сейчас все они используют так или иначе данный язык. И не надо думать, что JS только для оформления страниц, под его началом работает как клиентская, так и серверная часть интернет-приложений. Но, да, в основном он оспользуется для клиента.

Мощь внутри браузера.

JS - "безопасный" язык программирования, он не предоставляет на прямую доступ к памяти, где ведёт текущие расчёты он и другие программы. Потому он изначально и был создан для браузеров.

Как уже говорилось выше, JS способен на любое форматирование веб-страницами для взаимодействия с пользователем. Так, добавление, изменение, удаление блоков - для него обычное дело. Способность в редактирование стилей даёт возможность придать странице индивидуальный вид. А встроенные функции для обработки кнопок, значений помогут при интерактиве.

Но способен не на всё, а в пределах разумного. Так, запись, чтение файлов с жёсткого диска ему запрещена. Вам же не нужно, чтобы какой-то левый сайт узнал вашу личную информацию. Код одной страницы не может узнать о другой странице и её коде: опять же из-за безопасности, манипулирование другой вкладкой, другого сайта, например, личного кабинета банка, - странное и подозрительное мероприятие.

В данном примере используются два input - поле ввода и кнопка. При нажатии на кнопку срабатывает обработчик, настроенный на эту кнопку: 

                    dcmt.getElementById("formTask_0").addEventListener("submit", function (event) {

                            localStorage.setItem("Uname", document.getElementsByName("User")[0].value);

Используется объект document для обращения к форме, в которой расположены наши input. Метод addEventListener является прослушивателем нашей кнопки, при нажатии на которую срабатывает следующий код, где используется встроенный объект для локальной записи - localStorage - по ключу в нём сохраняется значение, введённое нами в поле с атрибутом name="User". Дальше идёт вызов метода printName(): 

                    let Name = "";
                    if (localStorage.getItem("Uname") == null) {
                        Name += "гость";
                    } else {
                        Name += localStorage.getItem("Uname");
                    }
                    document.getElementsByName("User").forEach(function (input) {
                        document.getElementById("usrName").textContent = "Привет, " + Name;
                    });

Здесь создаётся локальная переменная Name, её нельзя изменить за пределами данного метода. Конструкция if else - это условие, которое сначала проверяет правильность первого выражения, указанного в скобках, и соответственно выполняет действие в последующих фигурных скобках, а иначе, после else, запасное действие, которое срабатывает при ложном условии. Так, если в локальном хранилище отсутствует поле "Uname", то значит пользователь ещё не ввёл свое имя, и его приветствуют, как гостя. В противном случае локальной переменной присваивается сохранённое значение, которое добавляется в поле тега p с атрибутом name="usrName" справа вверху страницы.

Конструкции if else могут быть вложены друг в друга много раз, напрмер, 

if(...){
    ...
}else{
    if(){
        ...
    }else{
        ...
    }
}

Чтобы избежать таких громоздких конструкций, существет конструкция switch:

                    switch(значение){
                        case значение_1: действие_1;
                        break;
                        case значение_2: действие_2;
                        break;
                        ...
                        default: действие_по_умолчанию;
                        break;
                    }

Так, если мы заранее знаем какие значения попадутся, то можно воспользоваться данной конструкцией, которая найдёт наше значение и выполнит соответствующее действие. Ключевое слово break нужно для того, чтобы после выполнения действия работа конструкции завершилась, а не была продолжена. Действие default выполнится тогда, когда данного значения не было обнаружено в switch.

В JS это называется "Конкатенацией", с помощью оператора "+" можно складывать строки из переменных и строк, и сработает это в том случае, если есть хотя бы одна строка.                     console.log("Результат: "+(5*2+3))  //Результат: 13

В следующем примере используется уже знакомая конструкция if else. Тут сравниваются длины двух строк. 

                        if (str1.length == str2.length) {
                            dcmt.getElementById("TrOrFl").textContent = "true";
                        } else {
                            dcmt.getElementById("TrOrFl").textContent = "false";
                        }

Выражение условия сводится к простому "да" и "нет" или так называемым булевым значениям "true" "false". Большую роль здесь играет оператор, который мы используем для сравнения и который и возвращает значение "true" "false". В данном случае "==" сравнивает на равенство левое и правое значения. lenght - встроенное значение строк, которое само подсчитывает количество символов в строке. Существуют и другие операторы: "!=" сравнивает на неравенство, "<", ">" - меньше, больше, "<=" , ">=" меньше или равно, больше или равно, "&&" - левое и правое выражения должны возвращать либо истину, либо ложь, "||" - если хотя бы одно вернёт истину, то значит истина. Стоит отметить, что любое число или не пустая строка будут возвращать истину, а ноль и "" (пустая строка) вернут ложь.

                        console.log(!!5) //true
                        //"!!" приводит значения к булевскому формату
                        console.log(!!0) //false
                        console.log(1!=4) //true
                        console.log(1>4) //false

Здесь создаётся не просто переменная, а массив переменных, в котором по индексу можно хранить переменные. Массив характеризуется квадратными скобками. При инициализации (создании) массива в скобках можно указать начальные значения, сейчас массив пуст. Мы его заполняем через конструкцию for - цикл. С помощью циклов можно выполнить одно или несколько разных действий много раз. Первым параметром в нём указывается переменная, которая будет служить счётчиком для цикла, втором - условие, при котором цикл будет работать, и третьим - логика увеличения переменной-счётчика. i++ означает, что переменная увеличится на единицу после одного прохода по циклу. 

                    let arr = [];
                    let copy = 0;
                    for (let i = 0; i < 5; i++) {
                        arr[i] = dcmt.getElementsByName("Num_" + i)[0].value;
                    }
                    dcmt.getElementById("Min").textContent = "Min: " + Math.min.apply(null, arr);
                    dcmt.getElementById("Max").textContent = "Max: " + Math.max.apply(null, arr);

Math ещё один встроенный объект JS, в котором находятся большинство математических методов и переменных. В данном примере методы min и max находят минимум и максимум в массиве arr.

Помимо for существуют и другие виды циклов:

for (индекс-переменая in массив) {...} предназначен для перебора массива. В переменную присваивается индекс элемента массива.

for (переменная of массив){...} в переменную присваивается значение элемента массива.

while(условие){...} выполняется до тех пор, пока условие истинно.

Для циклов существуют два специальных оператора continue и break, первый прервёт действие и пойдёт по циклу следующим шагом, а второй завершит весь цикл, даже если условие в начале цикла истинно.

В основе этого таймера лежит встроенный метод setInterval(func,ms). Данный метод выполняет действие func через определённое количество времени, указанное в милисекундах ms. В данном случае каждую секунду выпоняется логика таймера.

                    const timeID = setInterval(
                        () => {
                            //остановка при достижении 0 или паузы
                            if ((this.hh <= 0 && this.mm <= 0 && this.ss <= 0) ||
                                this.Button == false) {
                                clearTimeout(timeID);
                                dcmt.getElementsByName("HH")[0].disabled = false;
                                dcmt.getElementsByName("MM")[0].disabled = false;
                                dcmt.getElementsByName("SS")[0].disabled = false;
                            } else {
                                this.ss--;
                                if (this.ss < 0) {
                                    this.mm--;
                                    this.ss = 59;
                                }
                                if (this.mm < 0) {
                                    this.hh--;
                                    this.mm = 59;
                                }
                            }
                            this.showTime();
                            //при достижении 0
                            if (this.hh <= 0 && this.mm <= 0 && this.ss <= 0) {
                                dcmt.getElementsByName("TimeButton")[0].value = "Старт"
                                dcmt.getElementById("timerMessage").textContent = "Время прошло.";
                            }
                        }, 1000
                    );

Метод присвоен переменной с типом const. Такие переменные нельзя изменить в последующей работе. Приваивается он (метод) для того, чтобы потом его можно было прервать через внутреннее условие методом clearTimeout(переменная). Мы же не хотим, чтобы таймер работал вечность, а с данной логикой - ушёл в минус.

Метод setTimeout(func,ms) вызывает действие func всего один раз через определённое время. Для таймера он бы не подошёл.

Конструкция ()=>{...} - это так называемая стрелочная функция, когда нужно передать какую-либо логику в какой-либо метод, вызывается она. В круглых скобках можно указать аргументы, с которыми будет работать тело этой функции.

Здесь для вызова одной из трёх картинок используется генерация значения от 0 до 3 без последнего. Используются два метода известного нам уже объекта Math.

let rand=Math.floor(Math.random()*3);

random() генерирует вещественное значение от 0 до 1 без последней. floor() округляет значение до наименьшего целого. И если вызвать Math.floor(Math.random()), нам вернётся 0. Поэтому нужно умножить наше вещественное число на определённое значение.

Для даты создаётся объект с помощью ключевого слова new.


                let d=new Date();
                dcmt.getElementById("date").textContent=d.getDate()+"."
                +(d.getMonth()+1)+"."+d.getFullYear();

Теперь в переменной d хранится объект Date, и можно через неё вызвать методы для получения сегодняшней даты месяца, самого месяца и года. Все они возвращают целые значения, а метод месяца к тому же возвращает значение начиная с нуля подобно индексу массива, поэтому на это стоит обращать внимание при будущей разработке своего календаря.
