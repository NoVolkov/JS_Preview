Вступление

Здравствуй, дорогой читатель. Если ты читаешь это, то значит тебе интересно современное программирование. Здесь ты узнаешь основы JavaScript.

JavaScript - это ...

... один из самых популярных языков программирования, который заставляет выглядеть веб-страницы живыми. Раньше в давние времена эти страницы были статичными, плоскими, сейчас все они используют так или иначе данный язык. И не надо думать, что JS только для оформления страниц, под его началом работает как клиентская, так и серверная часть интернет-приложений. Но, да, в основном он оспользуется для клиента.

Мощь внутри браузера.

JS - "безопасный" язык программирования, он не предоставляет на прямую доступ к памяти, где ведёт текущие расчёты он и другие программы. Потому он изначально и был создан для браузеров.

Как уже говорилось выше, JS способен на любое форматирование веб-страницами для взаимодействия с пользователем. Так, добавление, изменение, удаление блоков - для него обычное дело. Способность в редактирование стилей даёт возможность придать странице индивидуальный вид. А встроенные функции для обработки кнопок, значений помогут при интерактиве.

Но способен не на всё, а в пределах разумного. Так, запись, чтение файлов с жёсткого диска ему запрещена. Вам же не нужно, чтобы какой-то левый сайт узнал вашу личную информацию. Код одной страницы не может узнать о другой странице и её коде: опять же из-за безопасности, манипулирование другой вкладкой, другого сайта, например, личного кабинета банка, - странное и подозрительное мероприятие.

В данном примере используются два input - поле ввода и кнопка. При нажатии на кнопку срабатывает обработчик, настроенный на эту кнопку: 

                    dcmt.getElementById("formTask_0").addEventListener("submit", function (event) {
                        event.preventDefault();
                            localStorage.setItem("Uname", document.getElementsByName("User")[0].value);
                        printName();
                    });

Используется объект document для обращения к форме, в которой расположены наши input. Метод addEventListener является прослушивателем нашей кнопки, при нажатии на которую срабатывает следующий код, где используется встроенный объект для локальной записи - localStorage - по ключу в нём сохраняется значение, введённое нами в поле с атрибутом name="User". Дальше идёт вызов метода printName():

                    let Name = "";
                        if (localStorage.getItem("Uname") == null) {
                            Name += "гость";
                        } else {
                            Name += localStorage.getItem("Uname");
                        }
                        document.getElementsByName("User").forEach(function (input) {
                            document.getElementById("usrName").textContent = "Привет, " + Name;
                        });

Здесь создаётся локальная переменная Name, её нельзя изменить за пределами данного метода. Конструкция if else - это условие, которое сначала проверяет правильность первого выражения, указанного в скобках, и соответственно выполняет действие в последующих фигурных скобках, а иначе, после else, запасное действие, которое срабатывает при ложном условии. Так, если в локальном хранилище отсутствует поле "Uname", то значит пользователь ещё не ввёл свое имя, и его приветствуют, как гостя. В противном случае локальной переменной присваивается сохранённое значение, которое добавляется в поле тэга p с атрибутом name="usrName" справа вверху страницы.

Конструкции if else могут быть вложены друг в друга много раз, напрмер, 

                    if(...){
                        ...
                    }else{
                        if(){
                            ...
                        }else{
                            ...
                        }
                    }

Чтобы избежать таких громоздких конструкций, существет конструкция switch:

                    switch(значение){
                        case значение_1: действие_1;
                        break;
                        case значение_2: действие_2;
                        break;
                        ...
                        default: действие_по_умолчанию;
                        break;
                    }

Так, если мы заранее знаем какие значения попадутся, то можно воспользоваться данной конструкцией, которая найдёт наше значение и выполнит соответствующее действие. Ключевое слово break нужно для того, чтобы после выполнения действия работа конструкции завершилась, а не была продолжена. Действие default выполнится тогда, когда данного значения не было обнаружено в switch.
